Validation

0.2.0

Here we compare the results of PAMLj with other software that performs power analysis. In particular, we will compare our results with R pwr package and G*Power.

Each software employs a z-test after arch transforming the proportions.

Example 1: Sample size

Setup

  • Aim = N
  • P1 = .6
  • P2 = .5
  • Power = .90
  • Alpha = .05
  • Tails = Two

PAMLj

The expected N is 1038, 519 per group.

R

(h<-pwr::ES.h(.6,.5))
## [1] 0.2013579
pwr::pwr.2p.test(h=h,power=.9,sig.level=.05)
## 
##      Difference of proportion power calculation for binomial distribution (arcsine transformation) 
## 
##               h = 0.2013579
##               n = 518.3089
##       sig.level = 0.05
##           power = 0.9
##     alternative = two.sided
## 
## NOTE: same sample sizes

ceiling the results we obtain 519 cases per group.

G*Power

Results are the same.

Example 2 : Power

Setup

  • Aim = Power
  • P1 = .6
  • P2 = .5
  • N per group = 200
  • Alpha = .05
  • Tails = Two

PAMLj

R

(h<-pwr::ES.h(.6,.5))
## [1] 0.2013579
pwr::pwr.2p.test(n=200,h=h,sig.level=.05)
## 
##      Difference of proportion power calculation for binomial distribution (arcsine transformation) 
## 
##               h = 0.2013579
##               n = 200
##       sig.level = 0.05
##           power = 0.5214145
##     alternative = two.sided
## 
## NOTE: same sample sizes

G*Power

Results are almost the same, with a trivial difference of -0.0087034

## NOTE: different sample sizes

G*Power

Again, results are the very similar.
